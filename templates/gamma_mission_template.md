# Gamma Mission Template

Missions are the high-level objectives in the neural expansion arcade.

## Sections
- **Protocol:** (Default: GAMMA)
- **Version:** (Semantic version)
- **Mode:** (scientific | debug)
- **Summary:** High-fidelity action trace and rationale.

### Scientific Mode
- **Research Question:** Hypothesis or data query.
- **Corpus Scope:** Papers, datasets, or codebases inspected.
- **Method Evidence:** Tools or logic used.
- **Uncertainty:** Known gaps or confidence levels.
- **Output Artifact:** Resulting Markdown or JSON.

### Debug Mode
- **Observed vs Expected:** Failure description and target state.
- **Reproduction:** Steps to trigger the error.
- **Root Cause:** Underlying reason for failure.
- **Patch Strategy:** Minimal-diff resolution approach.
