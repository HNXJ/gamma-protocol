# Gamma Templates

Standardized templates for Gamma artifacts. Use these as starting points for all formal documents.

| Template | Use For |
|---|---|
| mission_template.md | Defining new missions |
| receipt_template.md | Documenting execution receipts |
| event_template.md | Defining game events |
| tournament_template.md | Defining multi-agent competitions |
| judge_report_template.md | Documenting judge decisions |
| agent_handoff_template.md | Handing off work between agents |
| inventory_template.md | Setting up agent workspace |
| model_profile_template.md | Documenting model capabilities |
