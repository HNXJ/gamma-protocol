# Quest Contract

A quest is a bounded unit of scientific work.

## Structure
- **Objective:** Clear, falsifiable scientific goal.
- **Harness:** The specific harness required for execution.
- **Payoff:** The scoring rubric for success.
- **Dependencies:** Required prior world state or receipts.

## Completion
A quest is complete only when a valid receipt has been generated and a THETA validation report has been accepted.
