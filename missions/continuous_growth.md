# Mission Contract: Continuous Growth (N → N+1)

## Definition
Given a current validated network of N neurons, a Player proposes and adds one biologically justified neuron, excitatory or inhibitory, to produce an N+1 candidate network.

## Mission Parameters
- **Current State (N)**: 11
- **Target State (N+1)**: 12
- **Objective**: Expand the validated neural substrate while preserving stability and E/I balance.

## The City Metaphor
- The network is a growing city.
- Each neuron is a new inhabitant or infrastructure unit.
- Excitatory neurons expand signal propagation.
- Inhibitory neurons regulate timing, stability, and balance.

## Participant Roles
- **Players**: Propose candidate N+1 motifs, run trial simulations, and optimize parameters.
- **Judge**: Runs canonical validation on the N+1 candidate.

## Execution vs. Truth
- **Trial Simulations**: Scratch artifacts produced by Players. They do not affect truth and do not promote receipts.
- **Judge Validation**: Canonical, receipt-backed, and deterministic. PASS / REVISE / FAIL results.
- **Truth Rule**: Only a Judge-validated PASS with a receipt can commit N+1 as new truth.

## Staged-Growth Doctrine
### Early Game (Current)
- Preserve stability.
- Preserve E/I balance.
- Validate simple motifs.

### Mid Game
- Add inhibitory diversity.
- Add laminar structure.
- Add plasticity / receptor diversity.

### Late Game
- Add columns.
- Add areas.
- Add long-range projections.
- Add richer predictive-processing tasks.

## Scientific Caution
The long-term objective is to asymptotically increase biological, physiological, mathematical, and neuroscientific fidelity to brain-like computation, without claiming equivalence to an actual brain.

## The Loop
Once N+1 is committed, the mission automatically resets to (N+1) → (N+2).
