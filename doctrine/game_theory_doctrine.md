# Game-Theoretic Doctrine

Scientific discovery in Gamma is treated as a repeated game among bounded agents.

## Key Concepts
- **Bounded Agency:** Agents operate within finite resource and tool constraints defined by harnesses.
- **Payoff Surfaces:** Scoring rubrics define the incentives for agents, but scores are not truth.
- **Anti-Collusion:** Mechanisms are required to prevent agents from validating each other's claims without sufficient evidence.
- **Repeated Interaction:** Agents build reputation and capability over time through successive quests and audits.
- **Equilibrium vs. Consensus:** Progress is defined by receipt-backed evidence, not social agreement. Falsification is a core move in the game.
