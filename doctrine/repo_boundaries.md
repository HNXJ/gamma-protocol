# Repository Boundaries

- **gamma-protocol:** (The Doctrine) Markdown-only. Rules, contracts, schemas.
- **gamma:** (The Backend) Python/Orchestration. Solvers, adapters, truth gates, persistence.
- **gamma-arena:** (The UI) TypeScript/Vite. Dashboards, observation surfaces.
- **gamma-science:** (The Science) Scientific source code, manuscripts, and assets.
