# GAMMA: Guided Agentic Model-Mediated Actions

Welcome to the **GAMMA Protocol**. 

## 🧬 What is GAMMA?
GAMMA is a standard for **Cognitive Synchrony** in AI agents. Instead of simple chat logs, GAMMA produces **Guided Actions**—standardized traces that help multiple AI agents (like a Visual Engine and a Programming Agent) work together without losing track of the goal.

## 🛠 Skills vs. GAMMA (For Freshmen)
Think of this like a math class:
- **The Skill** is the **Formula** (e.g., $a^2 + b^2 = c^2$). You learn it once and keep it in your textbook (`skill.md`).
- **The GAMMA** is the **Solved Homework**. It shows the step-by-step work for a *specific* problem (`gamma.gamma`).

## 📏 The Rules of the Protocol
1. **The Chain:** Every Gamma must have a Problem and a Solution. No "floating" code without a reason.
2. **The Mediator:** The Main Model (the "Thalamus") must guide all agent hand-offs.
3. **The Delta:** Every Gamma must end in a real Action (a code change or a file update).

## 🚀 The Vision
Our goal is to integrate with the Gemini CLI so that in the future, you don't just report a bug; you **Submit a Gamma**.
