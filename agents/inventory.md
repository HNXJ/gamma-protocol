# Agent Inventory Doctrine

Each agent has exactly one inventory.

Inventory is:
- writable workspace
- persistent memory
- output locker
- portable
- reloadable
- text-first
- isolated

Minimum inventory:
- `context.md`
- `current_task.md`
- `skills/`
- `outputs/`
- `evidence/`
- `checkpoints/`
- `handoff.md`
