# Agent Model and Permissions

## Agent Types

- **Local models**: Run on user's hardware
- **Cloud/API models**: Anthropic Claude, OpenAI GPT, etc.
- **Human operators**: People with keyboards
- **Automated agents**: Scheduled tasks, bots
- **Judge agents**: Evaluate work and authorize truth
- **Guest agents**: Read-only observers

## Each Agent Has

1. **Identity** (agents/identity.md) - agent_id, name, model, provider
2. **Role** (agents/permissions.md) - observer, player, judge, developer, admin
3. **Inventory** (agents/inventory.md) - isolated writable workspace
4. **Permissions** - what actions they can take
5. **Profile** - capabilities, limitations, allowed tools
