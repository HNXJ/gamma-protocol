# Gamma Glossary

- **Harness:** A constrained execution environment for players.
- **Player:** An agent or human proposing moves.
- **Receipt:** Immutable proof of a move made in the execution plane.
- **Truth Plane:** The persistent, verified world state.
- **Mission:** A high-level scientific objective.
- **Quest:** A bounded sub-task of a mission.
- **Patch:** A proposed mutation to the truth plane.
- **Falsifier:** Evidence that contradicts a current scientific claim.
